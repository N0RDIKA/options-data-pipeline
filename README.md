# Options Data Processing Pipeline

Небольшой проект для автоматизации работы с данными по акциям и опционам.

Я написала пайплайн, который последовательно собирает данные через API, очищает их, объединяет и считает ключевые показатели, чтобы на выходе получить готовый датасет для анализа.

## Что происходит внутри

Скрипты запускаются последовательно через `main.py`.

1. Загружаются текущие цены акций
2. Загружаются данные по опционам
3. Данные объединяются (цена акции добавляется к каждому опциону)
4. Отбираются опционы в диапазоне ±40% от цены базового актива
5. Рассчитываются показатели:

   * внутренняя стоимость
   * временная стоимость
   * delta
   * дни до экспирации
6. Данные приводятся к удобному формату
7. Формируется итоговый CSV файл

## Зачем это нужно

Проект упрощает подготовку данных для анализа опционов:
вместо ручной выгрузки и обработки всё делается автоматически.

## Стек

Python, requests, работа с API, обработка JSON/CSV, базовая финансовая логика

## Как запустить

1. Установить зависимости:
   pip install requests

2. Создать файл `peremen.py` и указать:

* API ключ
* список тикеров

3. Запустить:
   python main.py

## Особенности

* последовательный пайплайн обработки
* раздельные этапы (удобно менять и тестировать)
* обработка ошибок
* учёт лимитов API
